HTML, CSS, dan JavaScript - Basic Explanation

HTML dan CSS adalah Markup Language, untuk mengatur komponen dan tampilan sebuah halaman Web Javascript adalah Bahasa pemrograman Web

HTML

HTML adalah inti dari setiap halaman web, terlepas dari kompleksitas situs atau jumlah teknologi yang terlibat. Ini adalah keterampilan yang sangat penting bagi setiap profesional web. Bahasa ini merupakan titik awal bagi siapa saja yang belajar cara membuat konten untuk web.

HTML adalah singkatan dari HyperText Markup Language. "Bahasa markah" berarti bahwa, daripada menggunakan bahasa pemrograman untuk menjalankan fungsi, HTML menggunakan tag untuk mengidentifikasi berbagai jenis konten dan tujuan yang dilayaninya pada halaman web.

CSS

CSS adalah singkatan dari Cascading Style Sheets. Bahasa pemrograman ini menentukan bagaimana elemen HTML sebuah situs web seharusnya muncul di bagian depan halaman. HTML menyediakan alat mentah yang diperlukan untuk menyusun konten di situs web. Di sisi lain, CSS membantu menata konten ini sehingga terlihat oleh pengguna sesuai dengan yang diinginkan. Bahasa-bahasa ini dipisahkan untuk memastikan situs web dibangun dengan benar sebelum diformat ulang.

JavaScript

JavaScript adalah bahasa yang lebih rumit daripada HTML atau CSS, dan tidak dirilis dalam bentuk beta sampai tahun 1995. Saat ini, JavaScript didukung oleh semua peramban web modern dan digunakan di hampir semua situs di web untuk fungsionalitas yang lebih kuat dan kompleks.

JavaScript adalah bahasa pemrograman berbasis logika yang dapat digunakan untuk memodifikasi konten situs web dan membuatnya berperilaku dengan cara yang berbeda sebagai respons terhadap tindakan pengguna. Penggunaan umum JavaScript meliputi kotak konfirmasi, ajakan bertindak, dan menambahkan identitas baru pada informasi yang sudah ada.

Singkatnya, JavaScript adalah bahasa pemrograman yang memungkinkan pengembang web mendesain situs interaktif. Sebagian besar perilaku dinamis yang akan Anda lihat di halaman web adalah berkat JavaScript, yang menambah kontrol dan perilaku default browser.

Bootstrap - Basic Explanation

Bootstrap adalah framework web development berbasis HTML, CSS, dan JavaScript yang dirancang untuk mempercepat proses pengembangan web responsive dan mobile-first (memprioritaskan perangkat seluler).
